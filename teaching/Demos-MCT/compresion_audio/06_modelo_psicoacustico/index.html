6. Modelo Psicoacústico

Umbral global de enmascaramiento y asignación de bits por banda

Dos máscaras y umbral global de enmascaramiento

Cada máscara tonal genera un patrón de enmascaramiento que se extiende asimétricamente: cae rápido hacia frecuencias bajas y más lento hacia altas (enmascaramiento hacia arriba). Donde el patrón cae por debajo del umbral absoluto de audición, la máscara desaparece — no aporta enmascaramiento adicional. El umbral global (blanco) es la envolvente máxima de todos los patrones más el umbral absoluto. Un codec como MP3 analiza el espectro de cada trama de audio, calcula este umbral global y descarta toda información que quede por debajo de él.

Cómo usa este umbral un codec MP3

Análisis por trama
MP3 divide la señal en tramas de ~26 ms y calcula la FFT de cada una. Las componentes espectrales de esa trama son las propias máscaras — el umbral global se recalcula en tiempo real para cada trama.
Asignación de bits
Los bits de cada banda son (96 dB − umbral global) / 6, donde 96 dB es el techo dinámico de un sistema de 16 bits. Cuanto más alto el umbral de enmascaramiento, menos bits necesita esa banda.
