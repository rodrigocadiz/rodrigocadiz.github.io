2. El Problema del Cocktail Party

Cherry (1953): separación de fuentes sonoras concurrentes

El problema

En una fiesta con mucho ruido de fondo, los humanos podemos seguir fácilmente una sola conversación ignorando las demás. Esto se llama el problema del cocktail party (Cherry, 1953). El sistema auditivo analiza la mezcla de ondas que llega a los oídos y logra separar las fuentes: ¿cuántas personas están hablando? ¿Dónde están? ¿Qué dice cada una? Activa y desactiva las "voces" para escuchar la diferencia entre una mezcla y las fuentes separadas.

Escena auditiva — activa/desactiva cada voz haciendo clic

Claves para la separación

El cerebro usa múltiples claves acústicas para separar fuentes:

Claves espectrales
• Diferencia de timbre (forma espectral)
• Diferencias en la frecuencia fundamental (pitch)
• Estructura armónica diferente por voz
Claves temporales
• Onset/offset asincrónicos: fuentes que empiezan juntas se fusionan
• Modulación de amplitud diferente por fuente
• Continuidad temporal de cada flujo
Claves espaciales (binaural)
• Diferencia interaural de tiempo (ITD): llega antes a un oído
• Diferencia interaural de nivel (ILD): más fuerte en un oído
• Localización espacial: ubica la fuente en el espacio
Analogía de Bregman
Como un lago con surcos: el sistema auditivo analiza el movimiento de dos "hojas flotantes" y deduce cuántos "botes" (fuentes) hay en el lago y dónde están — ¡sin ver el lago directamente!
