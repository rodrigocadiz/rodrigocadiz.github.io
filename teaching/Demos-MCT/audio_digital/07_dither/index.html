7. Dither

Ruido blanco para mejorar la cuantización de señales débiles

El problema: señal de baja amplitud con pocos bits

Cuando una señal de baja amplitud se cuantiza con pocos bits, el error de cuantización deja de ser aleatorio y adquiere estructura periódica: se vuelve correlacionado con la señal y genera armónicos distorsionados audibles. El dither consiste en añadir ruido blanco aleatorio de amplitud ±½ LSB antes de cuantizar. Esto rompe la estructura del error, convirtiéndolo en ruido de fondo uniforme, menos molesto que la distorsión armónica.

Sin dither — error estructurado

¿Por qué funciona el dither?

Sin dither
La señal de baja amplitud ocupa solo unos pocos niveles de cuantización. El error es periódico y correlacionado con la señal → distorsión armónica (sonido metálico, "granular"), muy molesta para el oído.
Con dither
El ruido blanco añadido hace que la señal "salte" entre niveles de forma aleatoria. El error se vuelve ruido blanco no correlacionado → se escucha como suave "sss" de fondo en lugar de distorsión estructurada.
