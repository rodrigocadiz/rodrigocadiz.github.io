5. Cuantización y Error

Aproximación de la amplitud a niveles discretos

Señal cuantizada y error de cuantización

La cuantización asigna cada muestra al nivel discreto más cercano. Con N bits hay 2N niveles. La diferencia entre la señal original y la cuantizada es el error de cuantización, acotado a ±½ nivel (±½ LSB).

4 bits → 16 niveles | Error máx: ±0.063

Distribución del error

Error uniforme
El error de cuantización tiene distribución uniforme entre −½ y +½ del paso de cuantización (LSB). A más bits, menor es el paso y menor el error máximo.
Ruido de cuantización
El error tiene estructura: depende de la señal original. En señales de baja amplitud con pocos bits, el error puede dominar y crear distorsión armónica audible.
