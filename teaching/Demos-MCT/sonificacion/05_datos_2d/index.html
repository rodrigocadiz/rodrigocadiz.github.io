5. Espacio de Datos 2D

Exploración interactiva de datos con sonido — mueve el cursor sobre el mapa

Exploración sonora interactiva

Mueve el cursor sobre el mapa. El valor del dato en cada posición (representado por el color) controla el sonido: zonas claras/cálidas suenan distinto a zonas oscuras/frías. La posición del cursor solo indica dónde estás en el espacio de datos — el sonido refleja qué valor hay ahí. Este es el principio de la sonificación interactiva.

Activa el sonido y mueve el cursor sobre el mapa

Sonificación interactiva

Principio
El usuario navega el espacio de datos con el cursor. El sonido responde en tiempo real, creando un canal paralelo de información al visual. El oído puede detectar estructuras que la vista no nota inmediatamente.
Dimensiones auditivas usadas
X → Pitch (110–1760 Hz, 4 octavas)
Y → Volumen (silencio → forte)
Color del mapa → valor Z de la función subyacente
