Notes

            

                
C D E F G A BMiddle-C octave

                
c d e f g a bOne octave up

                
c'Two octaves up

                
C, D,One octave down

            


            
Accidentals

            

                
^CC sharp

                
_BB flat

                
=CC natural

                
^^C   __BDouble sharp / flat

            


            
Durations

            

                
C2Double the unit length

                
C/2Half the unit length

                
C3/2Dotted (1.5× unit)

                
C44× unit length

            


            
Rests

            

                
zRest (unit length)

                
z2Rest (double)

            


            
Bar Lines

            

                
|Bar line

                
||Double bar

                
|:   :|Repeat start / end
